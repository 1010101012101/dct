<h1 style="text-align: center">
会议服务平台升级内容通知(2011-09-01)
</h1>

<p>
 
<br/>
会议服务平台2011-09-01进行了升级，主要变更内容如下：
<br/>

<br/>
1、注册界面增加了签证信息，方便会务人员收集参会代表的签证信息，填写国际合作局那个邀请函使用。该功能与原来的陪同人员信息合并在一起了。
<br/>

<br/>
2、问题/反馈功能进一步完善，增加了问题分类联系人。参会代表可以根据问题类型选择发送邮件给不同的负责人。管理员可以在后台管理中查看和回复。
<br/>

<br/>
3、投稿界面，摘要的附件改为可选项，可以不显示摘要附件，也可以显示为可选项。
<br/>

<br/>
4、注册配置中，个人信息字段，增加“是否显示”的属性。系统默认提供的15个字段不可删除，可以设置是否显示。用户自定义的注册字段，可以删除。
<br/>

<br/>
5、注册配置、稿件配置、酒店配置的界面，增加“预览”功能，以方便查看配置后的界面效果。
<br/>

<br/>
6、会议起始日期的配置，增加对会期为1天的支持。
<br/>

<br/>
7、个人中心的左菜单，根据功能是否开放和用户类型进行动态显示。比如，作者不会看到审稿的链接，只有评审专家和管理员可以看到。如果酒店预订功能关闭，参会代表在个人中心就不会看到酒店预订的链接。
<br/>

<br/>
 
</p>
