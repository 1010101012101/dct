解答用户在创建会议网站系统中遇到的各种问题
</span>
</p>

<p style="font-size: 1.2em;">
<strong>
会议创建与帐号管理
</strong>
</p>
            
<ol>
                
<li>
、关于注册开放，如果在注册配置时选择收费，则需要配置注册费后才能看到参会代表注册页面，不然系统默认为配置不完全，注册为关闭状态。
</li>
                
<li>
忘记密码，如何找回：如果参会代表在注册会议后，忘记了登录密码，则需要点击登录界面上“忘记密码”链接，然后在跳转页面输入邮件地址（即注册帐号）和验证码后，系统会发一封重置密码的邮件，用户打开邮件中修改密码的链接即可进行密码重置。
</li>
                
<li>
关于独立域名设置：当需要设置独立域名时，首先在“网站配置”中的查看网站IP地址，然后在域名服务器上将新域名指向此IP地址后，在“网站配置”中的网站域名处修改成新域名后保存即可。
</li>
                
<li>
关于页面排版（表格分块）
<br/>
                当设计的页面需要排版对齐显示时，可以在页面中插入一张表格，分块来填写各部分内容，编辑好内容后把表格的边框设置为0，这样保存后，既看不到表格，各部分的内容也可以分块对齐显示
</li>
                
<li>
如何做连接，当需要把一段文字制作成链接时，首先需要在编辑状态下，选中需要制作链接的文字，然后点击工具栏第一排的
<br/>
                
<br/>
                “Insert Link”来制作链接，链接的类型：1、当需要链接到一个已存在的页面，只需要输入“页面名称”后保存。2、当需要链接
<br/>
                
<br/>
                到一个外部链接，只需要输入链接的“URL”后保存。3、当需要制作成一个邮件链接，只需要输入“邮件地址”后保存。
</li>
            
</ol>
